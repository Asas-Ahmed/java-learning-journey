# Primitive Type

---

- for storing simple values
- numbers, characters, booleans

| Type   | Bytes  | Range   |
|--------|--------|---------|
| byte   | 1   | -128, 127  |
| short   | 2  | -32K, 32K  |
| int  | 4    | -2B, 2B   |
| long | 8 | |
| float | 4 |  |
| double | 8 |  |
| char | 2 | A,B,C |
| boolean | 1 | true / false |

```java
byte age = 20;
long viewsCount = 3_123_456_789L;
float price = 10.99F;
char letter = 'A';
boolean isEligible = false;
```
